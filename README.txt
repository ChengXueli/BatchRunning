一：各个文件说明
	1. 000RunMe.bat
		此文件为调用脚本，双击即可运行当前批次任务
	2. 000作业自动评测脚本.py
		此文件为执行任务的python脚本，000.bat也是调用该脚本
	3. input.txt
		该文件为运行的exe的输入参数，一个test case的输入参数占一行，可以用#作为注释
		如果没有参数，那么保持该文件为空即可

	4. job1_baselineX.exe
		此系列可执行文件为baseline的exe，每次运行任务时会首先执行baseline
	5. job1_学号.exe
		此系列文件为学生提交的exe，每次运行时会对单个学生的成绩生成PDF，并自动发送邮件给 "学号@uestc.edu.cn"

	6. wkhtmltopdf.exe
		该exe为生成PDF的可执行文件，可将其加入PATH变量中，或者放在当前路径下。注意：该exe不能重命名，必须保持当前名称。
		参考：https://wkhtmltopdf.org/downloads.html
	
二. 针对“000作业自动评测脚本.py”脚本的说明
	1. 使用前，需要填入使用者的学号和邮箱密码，密码需要到校园邮箱生成一次性密码。详见脚本中“Admin”和“Password”。
		也可以使用其他邮箱，比如QQ邮箱，但是需要填写完整的发送者QQ邮箱名，即(xxx@qq.com)，另外需要配置“Server_Addr”
	2. 任务号(即作业编号，如上所提到的“job1”)可在该脚本中指定，如果未指定，则采用当前文件所在目录名作为任务号. baseline和学生提交的exe名称必须以“任务号_xxx.exe”命名
	3. 每次运行任务，会自动生成以“任务号_当前时间”为名称的文件夹，该文件夹会生成运行日志和各个学生的成绩表，并且有一个成绩总表
	4. 该脚本支持定时任务，详见脚本中变量“Scheduled”
	5. 该脚本支持多进程运行，详见脚本中变量“Multi_Process”
